# cypress-cucumber-e2e
Experiment with Cypress and Cucumber for e2e tests.

This repo can be used as basic for QA engineers, which plan to use Cypress and Cucumber for e2e tests.

# Instructions

Please install before run:
```
node 14.19.1
pnpm 7
```

How to open cypress:
```
pnpm cy:open
```
